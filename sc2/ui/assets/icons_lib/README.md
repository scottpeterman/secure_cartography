# Icons Library

Place your network device icon files (JPG, PNG) in this directory.

The `platform_icon_map.json` in `../data/` maps device platforms to icon filenames.

## Expected Icons

Based on the default icon map, these icons are referenced:

### Switches
- `layer_3_switch.jpg` - Default L3 switch
- `workgroup_switch.jpg` - Access/workgroup switches  
- `mulitswitch_device.jpg` - Chassis switches (C9500, etc.)
- `Nexus_5000.jpg` - Arista switches
- `Nexus_7000.jpg` - Nexus 7000/9000
- `simulitlayer_switch.jpg` - Juniper QFX

### Routers
- `router.jpg` - Default router
- `voice_router.jpg` - Voice gateways (VG202, etc.)
- `7500ars_(7513).jpg` - Legacy routers (7206VXR)
- `carrier_routing_system.jpg` - Nexus routing

### Other
- `firewall.jpg` - Default firewall
- `ip_phone.jpg` - IP phones
- `ata.jpg` - Analog telephone adapters
- `wireless.jpg` - Wireless APs
- `pc.jpg` - Endpoints
- `sun_workstation.jpg` - Linux servers
- `storage_server.jpg` - Debian servers
- `generic_processor.jpg` - Unknown devices

## Icon Sources

Cisco and other vendor icons can be obtained from:
- Cisco Network Topology Icons: https://www.cisco.com/c/en/us/about/brand-center/network-topology-icons.html
- Or use your own icons matching the filenames above
